# Net

第一层：应用层。定义了用于在网络中进行通信和传输数据的接口；

第二层：表示层。定义不同的系统中数据的传输格式，编码和解码规范等；

第三层：会话层。管理用户的会话，控制用户间逻辑连接的建立和中断；

第四层：传输层。管理着网络中的端到端的数据传输；

第五层：网络层。定义网络设备间如何传输数据；

第六层：链路层。将上面的网络层的数据包封装成数据帧，便于物理层传输；
